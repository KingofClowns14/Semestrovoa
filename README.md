# Семестровая морской бой 
Разрабатывали студенты КБ-241 Курочкин Илья Андреевич, Россихина Арина Дмитриевна
Описание: проект представляет собой морской бой, в котором 2 клиента общаются между собой через сервер. Игровая доска отрисовывается в отдельном классе. Работает с JSON файлами, сами файлы представляют собой статистику побед и поражений игроков, файлы собираются вручную без помощи библиотек.
